Proyecto taller 2 POO, por Dante Pizarro y Antonio Arancibia, "El camino para ser el mejor", en este proyecto se creo un juego de simulacion de pokemon.
El cual a traves de un menu permite realizar varias acciones, entre ellas capturar pokemones, revisar tu equipo, desafiar lideres de gimnasio y altos mandos, 

Para este taller se utilizaron 6 clases, una clase pokemon que contiene los atributos de cada pokemon, una clase jugador que guarda los datos del jugador leido
desde un archivo, una clase lider y altomando, una clase para la tabla de tipos y finalemnte la clase Taller2 que funciona como clase App, que ejecuta el programa

Toda la informacion del programa se obtiene de archivos de textos, y la informacion del jugador se guardo en un archivo "Registros.txt" el cual se actualiza a medida
que cambia su equipo y sus medallas
